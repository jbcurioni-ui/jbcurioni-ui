## 🎯 Objetivo do Sistema
Este projeto foi desenvolvido para auxiliar no controle doméstico, permitindo calcular o consumo mensal de energia de diferentes aparelhos e estimar o custo em Reais (R$). É uma ferramenta simples e eficiente para conscientização sobre gastos elétricos.

## 🧮 Fórmula Utilizada
O cálculo do consumo segue a norma técnica básica para estimativa de kWh:

$$Consumo_{mensal} = \frac{Potência (W) \times Horas_{Uso} \times 30 dias}{1000}$$
